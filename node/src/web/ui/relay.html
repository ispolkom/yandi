ℹ️ Как это работает

1. Публичная нода (Relay Server)

Включите relay server на ноде с белым IP. Она будет ретранслировать трафик.

2. Нода за NAT (Relay Client)

Выберите пира из списка и нажмите "Подключиться". Relay создаст сессию и соединит вас через публичную ноду.

3. Результат

После подключения вы можете использовать SOCKS5 или HTTP Proxy через relay, даже если оба пира за NAT.
