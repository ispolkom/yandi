# Как помочь YANDI / How to Contribute

## Русский

YANDI — это открытый проект, и ты можешь помочь ему стать лучше.

### Чем можно помочь?

1. **Код** — открывай Pull Request с улучшениями
2. **Идеи** — создавай Issue с предложениями
3. **Тесты** — запускай, ищи баги, сообщай
4. **Документация** — улучшай README, переводи
5. **Популяризация** — расскажи о проекте другим

### Как помочь деньгами?

У меня нет донатов, криптокошельков и прочего. Это не коммерческий проект.

Если хочешь поддержать — **поставь звезду на GitHub** ⭐ и расскажи о проекте тем, кому это может быть интересно.

---

## English

YANDI is an open project, and you can help make it better.

### How can you help?

1. **Code** — open a Pull Request with improvements
2. **Ideas** — create an Issue with suggestions
3. **Tests** — run it, find bugs, report them
4. **Documentation** — improve README, translate
5. **Promotion** — tell others about the project

### How to support financially?

I don't have donations, crypto wallets, etc. This is not a commercial project.

If you want to support — **give a star on GitHub** ⭐ and tell others who might be interested.
